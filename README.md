# TypeScript Assignment Commands

## Setup

```bash
npm init -y
npm install typescript --save-dev
```

## Compile Any File

```bash
npx tsc <filename.ts>
```

## Run Any File

```bash
node <filename.js>
```

## Assignment Compile Commands

```bash
npx tsc Q1_StudentManagement.ts
```
